# MyFood

> 위치기반 식당 추천 앱
>
> 사용자끼리 그룹을 만들어 식당을 추천
>
> QR 코드를 생성하여 쉽게 그룹을 생성
> 
> 선호 집합에서 비선호 집합을 제거
